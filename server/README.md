# GTU Exam Buddy

A platform for GTU students to:

- Find old question papers
- Upload syllabus PDFs
- Analyze papers using AI
- Generate important questions
- Generate preparation PDFs

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT
- Refresh Tokens

### Storage
- Multer
- Cloudinary

### AI
- Gemini API
